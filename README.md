SDPM
System Requirements
a. The codes can be executed within the MATLAB environment and the R/RStudio environment on any operating system.
b. The codes were implemented using MATLAB R2022b and R 4.3.0.
c. No non-standard hardware is required.

Installation Guide
a. The MATLAB codes can be run directly without installation.
b. The required R packages are listed in the first few lines of the corresponding code files.

Instructions for Use
All datasets are provided in the “DATA” folder.
Step 1: Run “AS” to generate the file “rmse_station_all.mat”.
Step 2: Run “FS” to generate the file “combined_sd_y.mat”.
Step 3: Run “Main” to obtain the comparison plot of early warning signals and seismic events.

Evaluation Metrics
The performance of the early warning system is evaluated using the following metrics: Accuracy, Precision, Recall, F1-score, False Positive Rate (FPR), and False Negative Rate (FNR).

In addition, the codes for three other predictive models used for comparison are provided in the “Comparison” folder.
